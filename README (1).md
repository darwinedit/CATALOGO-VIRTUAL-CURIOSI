# Carpeta de fotos

Sube aquí las fotos de tus productos (botón "Add file" → "Upload files" en GitHub).

Luego, en `products.json`, en el campo "image" de cada producto, escribe:

    "images/nombre-del-archivo.jpg"

Ejemplo: si subiste "parlante-rojo.jpg", el campo queda así:

    "image": "images/parlante-rojo.jpg"

Consejos:
- Usa fotos cuadradas (1:1) para que se vean parejas en el catálogo.
- Si puedes, comprime las fotos antes de subirlas (por ejemplo con https://squoosh.app) para que la página cargue más rápido.
